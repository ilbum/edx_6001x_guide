Strings

Use single or double quotes to create a string

hi = "hello there"
greetings = 'hello'

# String concatenation
>>> name = "Eric"
>>> greet = hi + name
'hello thereEric'
>>> greeting = hi + " " + name
'hello there Eric'
>>> name*3
'EricEricEric'
>>> len('hi there')
8
